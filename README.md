`Mathematics:` Folders for solutions of _**MATHEMATICS**_ problems.

Data Structures, Mathematics_ Problems in HackerRank.

`Hard:` Contains solutions to _**HARD**_ problems, these includes - _Algorithms, 
Data Structures, Mathematics_ Problems in HackerRank.
